# Screenshot Guide

Store README screenshots in this folder and use stable, feature-based filenames.

## Naming convention

- `extension-<feature>.png`
- `app-<feature>.png`

Examples:

- `extension-options-badge-settings.png`
- `extension-youtube-watched-mark.png`
- `app-downloader-tab.png`
- `app-renamer-tab.png`
- `app-organizer-tab.png`

## Recommended capture checklist

### Extension
- YouTube page with WATCHED mark badge visible
- Extension Options - Badge settings
- Extension Options - Migration/Backup
- Extension Options - SQLite Sync

### App
- Downloader tab
- Renamer tab
- Organizer tab

## Image quality tips

- Use consistent window size (e.g., 1440x900)
- Keep browser/app language consistent for all shots
- Blur or mask personal information
- Prefer PNG for UI readability
